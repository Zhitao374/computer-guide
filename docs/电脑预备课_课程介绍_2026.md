# 给家长的一封信：孩子大学前的电脑预备课

> 您的孩子即将上大学。无论学什么专业，电脑都是最重要的学习工具。  
> 这门课不是教孩子“打游戏”或“当程序员”，而是让ta **会用、能解决问题、不被人坑**。  
> 12次课，每次1.5小时，假期学完，受益终身。

---

## 一、为什么您的孩子需要这门课？

### 1. 买电脑，不懂就会被坑
电脑城销售、网上低价机专骗不懂的学生。学完孩子能自己判断：
- 什么配置够用（不花冤枉钱）
- 什么配置是坑（不买电子垃圾）
- 在哪买、怎么验机（不被骗）

### 2. 电脑卡、慢、出问题，自己就能修
很多孩子只会重启，重启没用就发愁。学完后：
- 知道为什么卡（内存不够？C盘满了？）
- 会自己清理、关掉没用的程序
- 蓝屏、没声音、连不上网都能自己查原因

### 3. 大学作业全靠电脑，不会就耽误成绩
- 写论文：不会排版，格式乱，被扣分
- 做数据统计：别人1小时，ta弄一天
- 课堂展示：PPT难看，小组评分受影响  
学完后，这些都能独立完成，而且做得不差。

## 二、12天学完，孩子具体能做什么？（通俗版）

| 场景 | 孩子能做的事 |
|------|----------------|
| 买电脑 | 知道买什么配置不卡、去哪买安全、收到货会验机（不激活先检查） |
| 电脑卡慢 | 能判断是内存不够还是C盘满了；会关掉“偷跑”的程序；定期清理垃圾；知道“重启”为什么管用 |
| 文件管理 | 会分类存文件、压缩打包交作业；会用U盘和云盘双重备份，论文丢了能找回 |
| 写论文/作业 | 用Word排出带目录、页码、整齐排版的文章；会插图片、表格、公式 |
| 做数据/记账 | 用Excel自动算账；从一堆数据里筛选出有用的；会画柱状图、饼图 |
| 课堂展示 | 做出干净漂亮的幻灯片；加简单动画；导出PDF交给老师 |
| 上网找资料 | 会搜索技巧，找学习资料快又准；不下载流氓软件 |
| 安全问题 | 不点弹窗广告；会卸载捆绑软件；会备份重要文件 |

## 四、课程安排计划表（12次课，每次1.5小时）

| 课次 | 主题 | 核心内容 |
|------|------|----------|
| **第1次** | 认识电脑 + 鼠标键盘 | 硬件软件、开关机、鼠标5种操作、键盘打字基础、中英文切换、简体繁体切换 |
| **第2次** | 文件管理基础 | 文件/文件夹操作、扩展名对照表、复制/剪切/删除、回收站、固态vs机械硬盘 |
| **第3次** | 文件管理进阶 | 路径与搜索、快捷方式、压缩解压、批量重命名、文件系统与格式化 |
| **第4次** | Word论文排版基础 | 样式与目录、页眉页脚与分节、字体安装、格式刷与模板 |
| **第5次** | Word表格、图片与公式 | 表格制作、图片图文混排、数学公式、交叉引用、参考文献管理 |
| **第6次** | Excel数据整理与公式 | 自动填充、SUM/AVERAGE/MAX/MIN函数、相对/绝对引用、数据验证 |
| **第7次** | Excel排序、筛选与图表 | 排序筛选、数据透视表、柱形图/饼图、VLOOKUP函数入门 |
| **第8次** | PPT课堂展示 | 主题配色、排版原则、母版、简单动画（淡入/擦除）、切换效果、导出PDF（**精简实用，不教花哨动画**） |
| **第9次** | Word论文排版实战加练 | 综合实战：写一篇完整模拟论文，一次性走通全流程 |
| **第10次** | 浏览器与软件管理 | 标签页操作、高级搜索技巧、安全下载安装、系统更新、Defender杀毒 |
| **第11次** | 常见问题急救与维护 | 电脑变慢5步法、故障速查表、数据备份、SSD保养、电脑发热处理 |
| **第12次** | 效率工具与习惯 | 快捷键汇总、邮件礼仪、云存储、PDF处理、自学方法 |


## 五、课程怎么教？孩子能接受吗？

- **每次1.5小时**（两节课），假期12次，不累。
- **知识点正规讲解**，不糊弄。
- **幽默提醒防坑**（比如“删除桌面图标不等于卸载软件”），孩子笑着就记住了。
- **每个知识点都带动手练习**，不是光听不练。
- **扩展知识**：比如“运行内存和硬盘的区别”“固态硬盘为什么快”，让孩子不仅会用，还懂一点原理。
