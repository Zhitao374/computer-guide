## 📅 第12天：快捷键汇总、邮件、云存储、PDF、自学方法

### 一、📖 知识点

**1. 必背快捷键（打印贴屏幕旁）**
| 快捷键 | 作用 |
|--------|------|
| `Ctrl+C / X / V` | 复制/剪切/粘贴 |
| `Ctrl+Z / Y` | 撤销/重做 |
| `Ctrl+S` | 保存（随时按） |
| `Ctrl+F` | 查找关键字 |
| `Ctrl+A` | 全选 |
| `Alt+Tab` | 切换窗口 |
| `Win+D` | 显示桌面（再按恢复） |
| `Win+E` | 打开此电脑 |
| `Win+I` | 打开系统设置 |
| `Win+A` | 打开通知中心 |
| `Win+L` | 锁屏 |
| `F2` | 重命名 |
| `Delete` | 删除 |
| `Ctrl+Shift+Esc` | 任务管理器 |
| `Ctrl+Shift+T` | 浏览器恢复关闭的标签页 |
| `Win+Shift+S` | 截图 |

**2. 电子邮件（大学正式沟通）**
- 注册一个正式邮箱：建议 Outlook 或 学校邮箱（避免用QQ邮箱发正式邮件，易被当垃圾）。
- 邮件礼仪：
  - 主题明确：“张三-计算机作业-第二次实验报告”
  - 称呼：尊敬的X老师，您好。
  - 正文：简明扼要，讲清需求。
  - 附件：确认已添加，命名规范（不要“新建文档1.docx”）。
  - 签名：留下姓名、学号、联系方式。
- 发送前检查一遍。

**3. 云存储（防丢神器）**
- **推荐**：
  - 百度网盘：2TB免费，但下载限速。
  - 阿里云盘：不限速，免费100GB。
  - OneDrive：Windows集成，5GB免费，可买Office 365送1TB。
- **用法**：将“大学资料”文件夹同步到云盘，换电脑也能访问。

**4. PDF 处理（大学必备）**
- **阅读与标注**：Edge浏览器直接打开PDF，可高亮、添加注释。
- **合并/拆分PDF**：使用 `ilovepdf.com`（免费在线，无需安装）。
- **Word转PDF**：文件 → 另存为 → 选择 `.pdf`。
- **PDF转Word**：需要专业软件或在线工具（注意隐私，敏感文件别传网上）。

**5. 持续学习与自学能力**
- **遇到问题怎么办**：
  1. 读屏幕提示（很多弹窗已经写了原因）。
  2. 复制错误信息到搜索引擎（百度/必应）。
  3. 在B站/知乎搜索相似问题，看视频或图文。
  4. 问AI（如ChatGPT、DeepSeek），描述清楚现象。
  5. 问同学或学长（最后手段）。
- **推荐资源**：
  - B站UP主：**oeasy**（全系列免费）、**旁门左道PPT**、**王佩丰Excel**、**秋叶PPT**。
  - 网站：ExcelHome、Office官方帮助（按 `F1`）。
  - 公众号：软件安装管家、电脑大师。

### 二、💡 常用操作技巧
- `Win + .`（句点）：打开表情符号面板，可输入 Emoji 😊🎓。
- `Win + 数字键`：打开任务栏上第几个程序（如 `Win+1` 打开第一个）。
- 快速锁定电脑：`Win+L`，养成离开座位就按的习惯。

### 三、🔍 扩展知识点：选择大学电脑的建议
- **文科/商科**：轻薄本，16GB内存 + 512GB SSD，CPU i5/R5以上，屏幕15.6寸或14寸。
- **理工科（编程/绘图）**：游戏本或全能本，16GB+内存，独显（RTX 3050以上），1TB SSD。
- **不要买**：4GB内存（卡死）、128GB硬盘（不够用）、赛扬/奔腾处理器（太弱）。
- 二手电脑也能用，但确保是SSD + 8GB以上内存。

### 四、⚠️ 幽默防坑指南

> 🧨 **坑**：给老师发邮件没有主题，老师以为是垃圾邮件删了。  
> 👉 **真相**：主题是邮件的“脸”，至少写“班级+姓名+事由”。

> 🧨 **坑**：只把作业存U盘，U盘丢了，作业也没了。  
> 👉 **真相**：U盘 + 云盘 + 电脑本地，三处备份才是真安全。

> 🧨 **坑**：大学四年只会用“一指禅”打字，毕业论文打了三天三夜。  
> 👉 **真相**：现在开始练盲打，每天10分钟，一个月后速度翻倍。
