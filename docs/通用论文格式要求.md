
# 本科毕业论文格式要求（完整版）

> 基于 **GB/T 7713-1987《科学技术报告、学位论文和学术论文的编写格式》** 及 **GB/T 7714-2015《信息与文献 参考文献著录规则》** 整理，各校可能有细微差异，**以你所在学校教务处发布的模板为准**。

---

## 一、页面与排版基础设置

| 项目 | 要求 |
|------|------|
| 纸张大小 | A4（210mm × 297mm） |
| 页边距 | 上 2.54cm，下 2.54cm，左 3.17cm（装订线留宽），右 2.54cm |
| 装订方式 | 左侧装订，装订线外留 0.5~1cm |
| 行距 | 正文一般 1.25倍 或 固定值 20~22磅 |
| 页眉 | 一般标注学校名称/论文题目，小五号宋体，居中或左对齐 |
| 页脚 | 页码居中，阿拉伯数字（封面不编页码，从摘要开始用罗马数字 i, ii… 正文起用阿拉伯数字 1, 2…） |
| 正文字体 | 中文：宋体（小四号）；英文/数字：Times New Roman（小四号） |
| 一级标题 | 黑体，三号，加粗，段前段后各 0.5 行 |
| 二级标题 | 黑体，四号，加粗，段前段后各 0.25 行 |
| 三级标题 | 黑体，小四号，加粗 |
| 四级标题 | 宋体，小四号，加粗 |

---

## 二、论文结构与各部分格式

### 📌 顺序一览

```
封面 → 原创性声明 → 版权使用授权书 → 摘要（中英文）→ 关键词 → 目录
→ 正文（引言→主体→结论）→ 参考文献 → 致谢 → 附录（如有）
```

---

### 1️⃣ 封面

| 要素 | 格式 |
|------|------|
| 论文题目 | 黑体，二号，居中，一般不超过 20 字 |
| 副标题（如有） | 破折号后接，楷体，三号 |
| 学院/专业 | 宋体，小四号，居中 |
| 学生姓名 | 宋体，小四号，居中 |
| 学号 | 宋体，小四号 |
| 指导教师 | 宋体，小四号 |
| 完成日期 | 年月，居中 |

> 封面一般不编页码。

---

### 2️⃣ 原创性声明 & 版权使用授权书

- 另起页，标题"原创性声明"用黑体三号居中
- 正文宋体小四，1.5倍行距
- 本人签名、日期手写或打印后手签
- 版权页同理

---

### 3️⃣ 摘要（Abstract）

| 项目 | 要求 |
|------|------|
| 中文摘要 | 标题"摘 要"黑体三号居中，两字间空一格 |
| 正文字体 | 宋体小四，200~300 字 |
| 内容要素 | 目的、方法、结果、结论（四要素必须齐全） |
| 关键词 | 3~5 个，"关键词："黑体小四，词之间用分号隔开 |
| 英文摘要 | 标题"ABSTRACT"黑体三号居中 |
| 英文正文 | Times New Roman，小四号 |
| 英文关键词 | Keywords: 斜体，词首字母大写，分号隔开 |

> 摘要单独成页，页码用罗马数字标注。

---

### 4️⃣ 目录（Table of Contents）

| 级别 | 格式 |
|------|------|
| 一级标题 | 黑体小四，左对齐 |
| 二级标题 | 缩进 2 字符，宋体小四 |
| 三级标题 | 缩进 4 字符，宋体小四 |
| 页码 | 右对齐，用……"引导 |
| 标题 | "目 录"黑体三号居中，字间空一格 |

> 目录自动生成，不要手动敲空格对齐。

---

### 5️⃣ 正文

正文是核心，一般结构：

```
1 引言（或绪论）
2 ×××（理论基础/文献综述/研究方法 等）
3 ×××（实验/调研/设计/分析）
4 ×××（结果与讨论）
5 结论与展望
```

#### 各级标题编号规则（推荐阿拉伯数字分级）

| 级别 | 编号格式 | 示例 |
|------|----------|------|
| 一级 | 1 2 3 … | 1 引言 |
| 二级 | 1.1 1.2 … | 1.1 研究背景 |
| 三级 | 1.1.1 1.1.2 … | 1.1.1 国内研究现状 |
| 四级 | 1.1.1.1 … | （一般不建议用到四级） |

> 编号后空一格接标题，**不要用顿号、不要加点**（如"1.研究背景" ✗ → "1 研究背景" ✓）

#### 正文中的其他格式要点

| 要素 | 要求 |
|------|------|
| 段落首行 | 缩进 2 字符 |
| 图表编号 | 图：图1-1、图1-2；表：表1-1、表1-2（按章编号） |
| 图标题 | 位于图**下方**，宋体五号居中，"图1-1 ×××" |
| 表标题 | 位于表**上方**，宋体五号居中，"表1-1 ×××" |
| 公式 | 居中，编号右对齐（如 (1-1)），用公式编辑器 |
| 引用文献标注 | 上标，如"……已有研究表明[1]"，或"张三（2020）认为……" |
| 数字用法 | 统计数据、编号用阿拉伯数字；约数、成语用汉字 |

---

### 6️⃣ 参考文献（References）

> 这是最容易出错的部分，严格按 **GB/T 7714-2015** 格式。

#### 排列顺序
按正文中引用的先后顺序编号（方括号 [1][2]…），不是按字母排。

#### 各类文献格式模板

| 类型 | 格式 | 示例 |
|------|------|------|
| **期刊** | [序号] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码. | [1] 张三, 李四. 深度学习在图像识别中的应用[J]. 计算机学报, 2023, 46(3): 45-58. |
| **专著（书）** | [序号] 作者. 书名[M]. 版次(第1版不写). 出版地: 出版社, 年: 页码. | [2] 王五. 机器学习导论[M]. 2版. 北京: 清华大学出版社, 2021: 120-135. |
| **学位论文** | [序号] 作者. 题名[D]. 保存地: 保存单位, 年. | [3] 赵六. 基于CNN的目标检测研究[D]. 成都: 电子科技大学, 2022. |
| **会议论文** | [序号] 作者. 题名[C]// 论文集名. 出版地: 出版社, 年: 页码. | [4] 孙七. A novel approach to NLP[C]// Proc of ACL 2023. Toronto: ACL, 2023: 201-210. |
| **网页/电子资源** | [序号] 作者. 题名[EB/OL]. (更新日期)[引用日期]. URL. | [5] 教育部. 本科毕业论文管理规定[EB/OL]. (2024-01-15)[2025-06-01]. http://xxx. |

#### 关键规则

| 规则 | 说明 |
|------|------|
| 作者 | 3人以内全列，超过3人列前3人 + "等"（英文用 et al.） |
| 标点符号 | 全部用英文半角标点（., : [ ]） |
| 不要写 | "参考文献如下：""详见文献[1]"等多余文字 |
| 数量 | 本科一般要求 **15~30篇**，其中近5年文献不少于 1/3，外文不少于 3~5 篇 |

---

### 7️⃣ 致谢

| 项目 | 要求 |
|------|------|
| 标题 | "致 谢"黑体三号居中，字间空一格 |
| 正文 | 宋体小四，1.5倍行距 |
| 内容 | 感谢导师、同学、家人、资助项目等 |
| 态度 | 真诚简洁，不要过于冗长，一般 300~500 字 |

---

### 8️⃣ 附录（如有）

| 项目 | 要求 |
|------|------|
| 标题 | "附录A ×××"黑体三号居中 |
| 编号 | 多个附录按 A、B、C… 顺序 |
| 内容 | 问卷、原始数据、代码、补充图表等不宜放正文的材料 |
| 格式 | 与正文一致 |

---

## 三、常见扣分项（避坑清单）

| ❌ 高频错误 | ✅ 正确做法 |
|-------------|-------------|
| 目录页码与正文不一致 | 用 Word 自动生成目录，更新域 |
| 参考文献正文中有、末尾缺 | 逐一核对，确保一一对应 |
| 图表无编号或编号不连续 | 用"交叉引用"功能，不要手动编号 |
| 中英文标点混用 | 中文用全角，英文/数字用半角 |
| 页眉页脚格式左右不一致 | 奇偶页不同的学校注意分节设置 |
| 摘要字数不够/关键词少于3个 | 中文摘要≥300字，关键词3~5个 |
| 正文抄袭率超标 | 一般要求 ≤30%（部分学校≤20%），务必查重 |
| 装订顺序错误 | 按学校要求排列，一般：封面→声明→摘要→目录→正文→参考文献→致谢→附录 |

---

## 四、字数要求（参考）

| 学科类型 | 一般要求 |
|----------|----------|
| 理工科 | 8000~15000 字（不含参考文献和附录） |
| 文史哲 | 10000~15000 字 |
| 艺术/设计类 | 正文+设计说明，视学校而定 |

---

## 五、装订顺序总览

```
封面
原创性声明
版权使用授权书
中文摘要 + 关键词
英文摘要 + 关键词
目录
正文（第1章 ~ 结论）
参考文献
致谢
附录（如有）
```
