# 第二阶段 大学必备办公软件（第4-9天）
## 📅 第4天：Word 论文排版基础（样式、目录、页眉页脚）

### 一、知识点
#### 1. Word 基础操作（必修）
- 新建文档：`Ctrl + N`
- 保存文档：`Ctrl + S`（高频必用）
- 字体快捷键：`Ctrl+B`加粗、`Ctrl+I`斜体、`Ctrl+U`下划线
- 段落对齐：`Ctrl+E`居中、`Ctrl+L`左对齐、`Ctrl+J`两端对齐
- 通用格式：正文**首行缩进2字符**，右键段落即可设置；高校论文通用字体：宋体。

#### 2. 样式功能（论文核心 必修）
- 样式：预设好的格式集合，分为标题1、标题2、正文等
- 使用方法：选中文字 → 开始选项卡 → 选择对应样式
- 优势：统一格式、一键生成目录，**禁止手动逐个修改字号加粗**。

#### 3. 自动目录（必修）
前提：文档已正确使用「标题1/标题2」样式
操作：引用 → 目录 → 选择自动目录
更新目录：右键目录 → 更新域 → 选择更新页码/全部更新

#### 4. 页眉页脚 + 基础页码（必修）
- 插入 → 页眉/页脚，可填写论文名称、班级姓名
- 插入 → 页码 → 页面底端居中，全篇连续编号（大一作业通用）
- 🟡 选学：分节符、不同章节不同页码（毕业论文高阶用法，用到再自学）

#### 5. 🟡 选学：字体安装（特殊格式要求时使用）
下载 `.ttf/.otf` 字体文件 → 双击安装 → 重启Word生效。

### 二、易错防坑指南
> 🧨 误区：用空格对齐标题、编号
> 👉 解答：空格宽度不固定，打印后排版错乱，使用段落/制表位对齐。
> 🧨 误区：目录页码错误，手动修改数字
> 👉 解答：手动修改会失效，必须右键更新目录。

### 三、今日课后练习（12分钟 必做）
1. 新建Word文档，设置2个一级标题、2个二级标题（使用样式）
2. 插入自动目录、添加页眉和页码
3. 保存文件

## 📅 第5天：Word 表格、图片与公式

### 一、知识点
#### 1. 表格制作（必修，课程表/成绩表常用）
- 插入 → 表格，拖动选择行列数
- 常用操作：右键**合并单元格**、手动调整列宽
- 重复表头（长表格分页必备）：选中表头行 → 布局 → 重复标题行

#### 2. 图片与图文混排（必修）
- 插入 → 图片 → 此设备，选择本地图片
- 文字环绕：右键图片 → 文字环绕，选择**四周型/紧密型**，图片可自由拖动
- 图片裁剪：选中图片 → 图片格式 → 裁剪
- 快捷截图粘贴：`Win+Shift+S` 截图后，`Ctrl+V` 直接粘贴到文档

#### 3. 公式插入（必修，理科作业/论文使用）
插入 → 公式，选择内置公式或手动输入，矢量格式清晰不模糊
- 🟡 选学：交叉引用（图表/公式引用，高阶论文使用）

### 二、易错防坑指南
> 🧨 误区：图片插入后无法移动、挤乱文字
> 👉 解答：默认是嵌入型，修改为四周型环绕即可。
> 🧨 误区：把公式做成图片插入，排版错位
> 👉 解答：使用Word自带公式工具。

### 三、今日课后练习（10分钟 必做）
1. 插入3×4表格，制作简易课程表
2. 插入一张图片，设置四周型环绕并调整大小
3. 插入一个简单数学公式

## 📅 第6天：Excel 数据整理与基础公式

### 一、知识点
#### 1. Excel 基础概念
- 工作簿：整个Excel文件
- 工作表：底部Sheet1/Sheet2分页
- 单元格：行列交叉位置，命名如 A1、B2

#### 2. 自动填充（必修，批量输入神器）
选中单元格，拖动右下角**填充柄**，可批量生成数字、星期、日期等序列。

#### 3. 公式与常用函数（必修）
- 规则：所有公式**必须以 = 开头**
- 基础运算：`=A1+B1`、`=A1*数字`
- 核心函数
  - `SUM(数据范围)`：求和，例 `=SUM(A1:A10)`
  - `AVERAGE(数据范围)`：求平均值
  - `MAX/MIN`：求最大值/最小值
- 快捷求和：`Alt + =`

#### 4. 单元格格式（必修）
- 问题：输入`001`自动变成`1`
- 解决：选中单元格 → 右键设置单元格格式为**文本**，再输入内容
- 补充：可设置货币、日期等格式。
- 🟡 选学：绝对引用 `$`、`F4` 切换引用格式（复杂计算使用）

### 二、易错防坑指南
> 🧨 误区：单元格只显示公式，不显示计算结果
> 👉 解答：单元格格式为文本，改为常规格式，重新回车即可。

### 三、今日课后练习（12分钟 必做）
1. 录入5行生活费收支数据（日期、项目、金额）
2. 使用SUM计算总支出，AVERAGE计算平均支出

## 📅 第7天：Excel 排序、筛选、数据透视表与图表

### 一、知识点
#### 1. 数据排序（必修）
选中数据区域 → 数据 → 排序，**勾选「我的数据具有标题」**，避免表头被排序。

#### 2. 数据筛选（必修）
选中标题行 → 数据 → 筛选；快捷键 `Ctrl+Shift+L`，可按条件隐藏/显示数据。

#### 3. 表格快速转换
快捷键 `Ctrl+T`，一键美化表格并自带筛选功能。

#### 4. 数据透视表（必修，分类汇总神器）
选中数据 → 插入 → 数据透视表，拖拽字段实现分类统计（如按消费类别汇总金额）。

#### 5. 图表制作（必修，课堂汇报展示）
选中数据 → 插入，选择柱形图、饼图、折线图；可添加数据标签、修改标题颜色。
- 🟡 选学：VLOOKUP查找函数（经管类专业选修）

### 二、易错防坑指南
> 🧨 误区：排序后表头错乱
> 👉 解答：排序时未勾选“我的数据具有标题”，重新操作即可。
> 🧨 误区：透视表只显示计数，无法求和
> 👉 解答：原数据包含文本/空白，检查金额列确保全为数字。

### 三、今日课后练习（10分钟 必做）
1. 对生活费表格按金额降序排序，筛选出餐饮类支出
2. 制作数据透视表统计各类别总支出，生成饼图

## 📅 第8天：PPT 课堂展示制作

### 一、知识点
#### 1. 幻灯片基础（必修）
- 新建幻灯片：`Ctrl + M`
- 左侧缩略图：拖拽可调整页面顺序
- 版式选择：标题页、内容页、图片页等，匹配展示内容。

#### 2. 主题与排版（必修）
- 设计 → 选择内置主题，统一全篇字体与配色
- 排版三原则：对齐、对比、留白；**单页文字不超过5行**，拒绝满屏文字。

#### 3. 幻灯片母版（必修，批量修改样式）
视图 → 幻灯片母版，统一添加Logo、页码、页脚，修改后应用到全部页面。

#### 4. 动画与切换（必修 简约风格）
大学课堂要求简洁，**禁止花哨动画**
- 动画：仅使用「淡入、擦除」
- 页面切换：仅使用「淡出」
- 放映快捷键：`F5`从头放映，`Shift+F5`从当前页放映

#### 5. 文件导出（必修）
- 导出PDF：文件 → 导出 → 创建PDF（提交作业最稳妥，兼容所有设备）
- 导出图片：另存为JPEG/PNG格式

### 二、易错防坑指南
> 🧨 误区：动画过多、效果花哨，观感杂乱
> 👉 解答：课堂展示以清晰为主，只保留基础淡入淡出。
> 🧨 误区：PPT发微信老师无法打开
> 👉 解答：优先导出PDF提交，避免格式后缀异常。

### 三、今日课后练习（12分钟 必做）
1. 制作3页PPT：封面+2个内容页，套用主题
2. 添加基础淡入动画、页面淡出切换
3. 导出为PDF保存

## 📅 第9天：Word 论文综合实战 + 格式兼容

### 一、知识点
#### 1. 模拟论文结构（必修）
精简版结构（2页以内）：封面 → 目录 → 正文 → 表格/图片
要求：正确使用标题样式、页眉页码、重复表头、图文混排。

#### 2. 高频问题修复（必修）
- 目录页码错误：全选目录，右键更新整个目录
- 分节后页眉同步变化：取消「链接到前一节」再修改内容
- 表格跨页无表头：重新设置「重复标题行」

#### 3. 格式兼容（必修）
校内同时使用Office和WPS，排版可能错乱，最终作业**务必同时保存 `.docx` 原文件 + PDF文件**。

### 二、今日课后练习（15分钟 必做）
1. 制作2页精简模拟论文，覆盖样式、目录、表格、图片
2. 分别保存为docx和PDF格式，存入指定文件夹
