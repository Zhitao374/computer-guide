# 第二阶段：办公软件核心能力（第3-10天）


## 📅 第3天：Word排版基础（样式/目录/页眉）

### 一、📖 知识点

#### 1. Word 基础操作（必修）
- 新建文档：`Ctrl + N`
- 保存文档：`Ctrl + S`（高频必用）
- 字体快捷键：`Ctrl+B`加粗、`Ctrl+I`斜体、`Ctrl+U`下划线
- 段落对齐：`Ctrl+E`居中、`Ctrl+L`左对齐、`Ctrl+J`两端对齐
- 通用格式：正文**首行缩进2字符**（右键→段落设置）；高校论文通用字体：宋体。

#### 2. 样式功能（论文核心 必修）
- 什么是样式：预设好的格式集合，分为标题1、标题2、正文等
- 使用方法：选中文字 → 开始选项卡 → 选择对应样式
- 优势：统一格式、一键生成目录，**禁止手动逐个修改字号加粗**

#### 3. 自动目录（必修）
- 前提：文档已正确使用「标题1/标题2」样式
- 操作：引用 → 目录 → 选择自动目录
- 更新目录：右键目录 → 更新域 → 选择更新页码/全部更新

#### 4. 页眉页脚 + 基础页码（必修）
- 插入 → 页眉/页脚，可填写论文名称、班级姓名
- 插入 → 页码 → 页面底端居中，全篇连续编号（大一作业通用）
- 🟡 选学：分节符、不同章节不同页码（毕业论文高阶用法，用到再自学）

### 二、⚠️ 易错防坑指南

> 🧨 误区：用空格对齐标题、编号  
> 👉 解答：空格宽度不固定，打印后排版错乱，使用段落/制表位对齐。

> 🧨 误区：目录页码错误，手动修改数字  
> 👉 解答：手动修改会失效，必须右键更新目录。

## 📅 第4天：Excel数据整理与公式

### 一、📖 知识点

#### 1. Excel 基础概念
- 工作簿：整个Excel文件
- 工作表：底部Sheet1/Sheet2分页
- 单元格：行列交叉位置，命名如 A1、B2

#### 2. 自动填充（必修，批量输入神器）
- 选中单元格，拖动右下角**填充柄**，可批量生成数字、星期、日期等序列。
- 案例：输入1、2，选中拖拽 → 生成3,4,5…；输入“星期一”拖拽 → 星期二、星期三…

#### 3. 公式与常用函数（必修）
- 规则：所有公式**必须以 = 开头**
- 基础运算：`=A1+B1`、`=A1*数字`
- 核心函数：
  - `SUM(数据范围)`：求和，例 `=SUM(A1:A10)`
  - `AVERAGE(数据范围)`：求平均值
  - `MAX/MIN`：求最大值/最小值
- 快捷求和：`Alt + =`

#### 4. 单元格格式（必修）
- 问题：输入`001`自动变成`1`
- 解决：选中单元格 → 右键设置单元格格式为**文本**，再输入内容
- 补充：可设置货币、日期等格式。
- 🟡 选学：绝对引用 `$A$1`（拖动公式时固定不变）、`F4` 切换引用格式（用到再学）

### 二、⚠️ 易错防坑指南

> 🧨 误区：单元格只显示公式，不显示计算结果  
> 👉 解答：单元格格式为文本，改为常规格式，重新回车即可。

## 📅 第5天：Word表格/图片/公式

### 一、📖 知识点

#### 1. 表格制作（必修，课程表/成绩表常用）
- 插入 → 表格，拖动选择行列数
- 常用操作：右键**合并单元格**、手动调整列宽
- 重复表头（长表格分页必备）：选中表头行 → 布局 → 重复标题行

#### 2. 图片与图文混排（必修）
- 插入 → 图片 → 此设备，选择本地图片

- 图片裁剪：选中图片 → 图片格式 → 裁剪
- 快捷截图粘贴：`Win+Shift+S` 截图后，`Ctrl+V` 直接粘贴到文档

#### 3. 公式插入（必修，理科作业/论文使用）
- 插入 → 公式，选择内置公式或手动输入，矢量格式清晰不模糊
- 🟡 选学：交叉引用（图表/公式引用，高阶论文使用）

### 二、⚠️ 易错防坑指南

> 🧨 误区：图片插入后无法移动、挤乱文字

> 🧨 误区：把公式做成图片插入，排版错位  
> 👉 解答：使用Word自带公式工具。

## 📅 第6天：Excel排序/筛选/透视表

### 一、📖 知识点

#### 1. 数据排序（必修）
- 选中数据区域 → 数据 → 排序，**勾选「我的数据具有标题」**，避免表头被排序。

#### 2. 数据筛选（必修）
- 选中标题行 → 数据 → 筛选；快捷键 `Ctrl+Shift+L`，可按条件隐藏/显示数据。

#### 3. 表格快速转换
- 快捷键 `Ctrl+T`，一键美化表格并自带筛选功能。

#### 4. 数据透视表（必修，分类汇总神器）
- 选中数据 → 插入 → 数据透视表，拖拽字段实现分类统计（如按消费类别汇总金额）。

### 二、⚠️ 易错防坑指南

> 🧨 误区：排序后表头错乱  
> 👉 解答：排序时未勾选“我的数据具有标题”，重新操作即可。

> 🧨 误区：透视表只显示计数，无法求和  
> 👉 解答：原数据包含文本/空白，检查金额列确保全为数字。

## 📅 第7天：Word+Excel综合实战（首次融合）

### 一、📖 实战任务

> 🎯 **本课目标**：用Excel分析数据，用Word输出完整报告。这是第一次将两个软件结合使用，体验真实工作流。

**任务**：分析“9月生活费”数据，输出一份《个人月度消费分析报告》

**Step 1：Excel中处理数据（30分钟）**
1. 打开第4天/第6天制作的生活费表
2. 使用数据透视表，按“类别”统计总支出
3. 使用排序功能，找出支出最高的三个类别
4. 将透视表结果**复制**（Ctrl+C）

**Step 2：Word中撰写报告（40分钟）**
1. 新建Word文档，标题为《个人月度消费分析报告》
2. 粘贴透视表（Ctrl+V），调整表格样式
3. 写3段文字分析：
   - 本月总支出是多少？
   - 哪个类别花费最多？为什么？
   - 下月如何优化预算？
4. 添加页眉“生活费分析报告”和页码
5. 保存为 `姓名_消费分析报告.docx`

**Step 3：导出提交（10分钟）**
- 另存为PDF，命名为 `姓名_消费分析报告.pdf`

### 二、💡 核心收获
通过本次实战，你将理解：
- **Excel是用来“算”的**：整理、统计、分析数据
- **Word是用来“写”的**：呈现结论、表达观点
- **两者配合**：Excel算完，Word写出报告，这是大学小组作业的标准流程

## 📅 第9天：Excel图表 + VLOOKUP + 数据验证

### 一、📖 知识点

#### 1. 图表制作（必修，课堂汇报展示）
- 选中数据 → 插入 → 选择柱形图、饼图、折线图
- 添加数据标签：选中图表 → “+”号 → 数据标签
- 修改标题、颜色
- **应用场景**：第7天消费分析报告，可用饼图直观显示各类别占比

#### 2. VLOOKUP查找函数（必修，查数据神器）
- 用途：根据一个值，在另一个表格中查找对应的信息
- 语法：`=VLOOKUP(找什么, 在哪找, 返回第几列, 0)`
- 案例：根据“学号”查找对应的“姓名”或“成绩”
- 学习建议：经管类/理工科同学务必掌握，文科同学了解即可

#### 3. 数据验证（下拉列表）（必修，做问卷/填表必备）
- 用途：限制单元格只能输入指定的内容（如性别只能选“男/女”）
- 操作：选中单元格 → 数据 → 数据验证 → 允许“序列” → 来源输入 `男,女`
- 效果：单元格右侧出现下拉箭头，点击选择，无法手动输入其他内容
- **大学场景**：做问卷调查、数据收集时，用下拉列表保证数据统一

### 二、⚠️ 易错防坑指南

> 🧨 误区：VLOOKUP返回#N/A错误  
> 👉 解答：查找值在原数据中不存在，或两表数据格式不一致（如文本型数字 vs 数值型数字）。

> 🧨 误区：饼图数据标签重叠看不清  
> 👉 解答：右键数据标签 → 设置数据标签格式 → 选择“最佳匹配”或手动拖动位置。

## 📅 第10天：Word长文档实战 + 批注/修订

### 一、📖 知识点

#### 1. 长文档结构（必修）
- 封面 → 目录 → 正文（含图表） → 参考文献
- 要求：正确使用标题样式、页眉页码、重复表头、图文混排

#### 2. 批注与修订（必修，小组作业互评必备）
- **批注**：选中文字 → 审阅 → 新建批注（用于提问、建议，不修改原文）
- **修订**：审阅 → 修订 → 开启修订模式（所有修改会留下痕迹，对方可接受/拒绝）
- **大学场景**：
  - 老师批改论文：用批注写评语
  - 小组作业互评：用修订模式修改组员内容，对方一目了然

#### 3. 高频问题修复（必修）
- 目录页码错误：右键目录 → 更新域 → 更新整个目录
- 分节后页眉同步变化：取消「链接到前一节」再修改内容
- 表格跨页无表头：重新设置「重复标题行」

#### 4. 格式兼容（必修）
- 校内同时使用Office和WPS，排版可能错乱
- 最终作业**务必同时保存 `.docx` 原文件 + PDF文件**

### 二、⚠️ 易错防坑指南

> 🧨 误区：不知道谁改了什么，小组作业乱成一团  
> 👉 解答：开启修订模式，每个人改的地方都带姓名标记，清晰可查。

> 🧨 误区：批注太多不知道怎么回复  
> 👉 解答：在批注框中直接输入文字即可回复，右键批注可“解决”（表示已处理）。
