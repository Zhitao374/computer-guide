## 📅 第4天：Word 论文排版基础（样式、目录、页眉页脚）

### 一、📖 知识点

**1. Word 界面与基本操作**
- 新建文档：`Ctrl+N`，保存：`Ctrl+S`（养成肌肉记忆）。
- 字体：`Ctrl+B` 加粗，`Ctrl+I` 斜体，`Ctrl+U` 下划线。
- 段落：`Ctrl+E` 居中，`Ctrl+L` 左对齐，`Ctrl+J` 两端对齐。首行缩进2字符（右键→段落）。

**2. 样式（大学论文核心）**
- 什么是样式：预置的格式集合（标题1、标题2、正文等）。
- 如何使用：选中标题文字 → “开始”选项卡 → 样式区的“标题1”。**不要手动改字号加粗**。
- 好处：一键生成目录、自动编号、方便整体调整格式。

**3. 插入目录**
- 引用 → 目录 → 自动目录（前提是用了标题样式）。
- 更新目录：右键目录 → 更新域 → 只更新页码/更新整个目录。

**4. 页眉页脚与页码**
- 插入 → 页眉/页脚 → 输入内容（如论文题目）。
- 插入 → 页码 → 页面底端 → 居中。
- **不同章节不同页码**（如目录用罗马数字，正文用阿拉伯数字）：需要插入“分节符”（布局 → 分隔符 → 下一页），然后取消“链接到前一节”。

**补充：字体安装方法**
- **下载字体**：可从“字体天下”、“识字体网”等网站下载 `.ttf` 或 `.otf` 文件。
- **安装字体**：双击下载的字体文件 → 点击左上角“安装”按钮 → 重启Word即可使用。
- **大学场景**：某些课程要求特定字体（如论文要求宋体、Times New Roman），学会安装可避免格式不符。

### 二、💡 常用操作技巧
- `Ctrl + Enter`：快速分页（比敲回车省事）。
- 导航窗格：视图 → 导航窗格 → 可快速跳转标题。

### 三、🔍 扩展知识点：模板与格式刷
- **模板**：`.dotx` 文件，保存样式和默认设置。大学毕业论文通常有学校模板。
- **格式刷**：选中格式好的文字 → 单击格式刷 → 刷到其他文字；**双击**格式刷可多次使用。

### 四、⚠️ 幽默防坑指南

> 🧨 **坑**：手动敲空格来对齐标题或图表编号。  
> 👉 **真相**：用“段落”→“制表位”或表格对齐。空格在不同字体下宽度不同，打印出来歪的。

> 🧨 **坑**：目录页码不对，手动修改数字。  
> 👉 **真相**：右键目录 → 更新域 → 只更新页码。手动改下次刷新又没了。

## 📅 第5天：Word 表格、图片、公式与参考文献

### 一、📖 知识点

**1. 表格（课程表、对比分析）**
- 插入 → 表格 → 拖动行列（或插入表格对话框）。
- 常用操作：合并单元格（右键 → 合并单元格）、调整列宽、表格样式（设计选项卡）。
- 表头重复：选中标题行 → 布局 → 重复标题行（长表格分页后自动重复表头）。

**2. 图片与图文混排**
- 插入图片：插入 → 图片 → 此设备。
- 文字环绕：右键图片 → “文字环绕” → 选“四周型”或“紧密型”→ 图片可自由拖动。
- 图片裁剪：选中 → 图片格式 → 裁剪 → 拖拽裁剪框。

**3. 公式与编号**
- 插入 → 公式 → 内置公式（如二次方程、勾股定理）或手写输入。
- 公式编号：可以使用“插入题注”或手动右对齐制表位（知道即可）。

**4. 参考文献管理**
- 简单方法：手动输入，格式统一（如 [1] 作者.书名.出版社,年份）。
- 推荐工具：**Zotero**（免费，大学必备），可插入引用并自动生成参考文献列表。
- Word自带：引用 → 插入脚注/尾注。

### 二、💡 常用操作技巧
- 快速插入截图：`Win+Shift+S` 截图后，`Ctrl+V` 直接粘贴到Word。
- 表格文字居中：选中单元格 → 右键 → 单元格对齐方式 → 水平居中。

### 三、🔍 扩展知识点：交叉引用
- **交叉引用**：在正文中引用图表或公式的编号（如“如图1-1所示”）。
- 操作：插入 → 交叉引用 → 引用类型选“图表”或“公式”→ 选择对应标签。
- 好处：图表编号改变时，正文引用自动更新（不用手动改）。

### 四、⚠️ 幽默防坑指南

> 🧨 **坑**：图片插入后文字被挤到下一页，图片动不了。  
> 👉 **真相**：默认“嵌入型”图片像大字一样占一行。右键图片 → “文字环绕”选“四周型”，它就变“漂浮”了。

> 🧨 **坑**：公式用图片插入，结果排版时对不齐。  
> 👉 **真相**：用Word自带的“公式”工具，它是矢量对象，永远清晰且对齐。

## 📅 第6天：Excel 数据整理与公式（生活费管理）

### 一、📖 知识点

**1. Excel 基础**
- 工作簿（文件）、工作表（Sheet1/Sheet2）、单元格（A1, B2）。
- 输入数据：单击单元格 → 输入 → 按回车确认。

**2. 自动填充（批量输入神器）**
- 选中单元格右下角小黑十字（填充柄）→ 向下/右拖拽。
- 案例：输入1、2，选中拖拽 → 生成3,4,5…；输入“星期一”拖拽 → 星期二、星期三…

**3. 公式与函数**
- 以 `=` 开头：`=A1+B1`，`=A1*0.1`。
- 常用函数（点击 `fx` 或手打）：
  - `SUM(范围)`：求和，如 `=SUM(A1:A10)`。
  - `AVERAGE(范围)`：平均值。
  - `MAX(范围)` / `MIN(范围)`：最大值/最小值。
- 快捷求和：`Alt + =`（自动写SUM公式）。

**4. 单元格引用**
- **相对引用**（默认）：拖动公式时行列自动变化。
- **绝对引用**（加 `$`）：如 `=$A$1`，拖动时不变。按 `F4` 可快速切换引用类型。

### 二、💡 常用操作技巧
- 快速选中数据区域：选中一个单元格 → `Ctrl + Shift + 方向键`。
- 调整列宽：双击列标（A、B、C…）之间的分隔线，自动适应内容。
- 设置单元格格式：右键 → 设置单元格格式 → 可设“货币”“日期”“文本”等。

### 三、🔍 扩展知识点：相对/绝对引用实战
- **计算比例**：每个项目支出 ÷ 总支出。总支出单元格需要绝对引用（`$B$10`），拖动公式时才不变。

**补充：数据验证/下拉列表（做问卷、统计时常用）**
- **用途**：限制单元格只能输入指定的内容（如性别只能选“男/女”），避免数据混乱。
- **操作**：选中要设置的单元格区域 → “数据”选项卡 → “数据验证” → 在“允许”中选择“序列” → 在“来源”中输入选项（用英文逗号分隔，如 `男,女`）→ 确定。
- **效果**：单元格右侧出现下拉箭头，点击可选，无法手动输入其他内容。

### 四、⚠️ 幽默防坑指南

> 🧨 **坑**：输入 `001`，回车后变成 `1`。  
> 👉 **真相**：Excel默认去掉前导零。解决方案：先选中单元格，设格式为“文本”，再输入。

> 🧨 **坑**：公式显示为 `=A1+B1` 而不是计算结果。  
> 👉 **真相**：单元格格式被设成了“文本”。改为“常规”，然后双击公式单元格回车。

### 五、✅ 今日练习（45分钟）

## 📅 第7天：Excel 排序、筛选、透视表与图表（大学数据分析）

### 一、📖 知识点

**1. 排序**
- 选中数据区域 → “数据”选项卡 → “排序” → 主要关键字（如金额）→ 升序/降序。
- 注意：勾选“我的数据具有标题”避免表头被排序。

**2. 筛选**
- 选中标题行 → “数据” → “筛选”（标题出现下拉箭头）。
- 点击下拉箭头 → 数字筛选 → 大于100 → 只显示金额>100的行。
- **快捷键**：`Ctrl+Shift+L` 快速开/关筛选。

**3. 数据透视表（大学神器）**
- 用途：快速分类汇总、交叉分析。
- 操作：选中数据区域 → “插入” → “数据透视表” → 新工作表。
- 拖拽字段：将“类别”拖到“行”区域，将“金额”拖到“值”区域 → 自动汇总各类别支出。
- 可继续拖拽“日期”到“列”区域，看每日各类别支出。

**4. 图表**
- 选中数据（不含总计行）→ “插入” → 图表类型（柱形图、饼图、折线图）。
- 添加数据标签：选中图表 → “图表设计” → “添加图表元素” → “数据标签”。
- 修改标题、颜色等。

### 二、💡 常用操作技巧
- `Ctrl+T`：将数据区域转为“表格”，自动美化并自带筛选和总行。
- 快速去重：数据 → 删除重复值（保留唯一记录）。

### 三、🔍 扩展知识点：VLOOKUP 函数（大学作业高频）
- 用途：根据一个值在另一个表格中查找对应信息。
- 语法：`=VLOOKUP(查找值, 表格区域, 返回第几列, 0)`。
- 案例：根据学号查找学生姓名。
- 学习建议：先掌握基础，大一下学期再深入。

### 四、⚠️ 幽默防坑指南

> 🧨 **坑**：排序后发现表头也被排进去了（变成最后一行）。  
> 👉 **真相**：排序时没勾选“我的数据具有标题”。重来。

> 🧨 **坑**：数据透视表里的数字无法求和，显示“计数”。  
> 👉 **真相**：金额列中有空白或文本。检查原数据，确保金额列全是数字。

## 📅 第8天：PPT 课堂展示（精简实用）

> 🎯 **大学PPT原则**：整洁、清晰、能导出PDF即可。花哨动画老师反而嫌弃，本课只教真正用得上的。

### 一、📖 知识点

**1. 幻灯片基本操作**
- 新建幻灯片：`Ctrl+M` 或“开始”→“新建幻灯片”。
- 版式：选择适合的版式（标题页、内容页、图片页等）。
- 移动幻灯片：左侧缩略图拖拽排序。

**2. 主题与配色**
- “设计”选项卡 → 选择内置主题（统一字体和配色）。
- 背景格式：右键幻灯片空白 → “设置背景格式” → 纯色/渐变/图片。

**3. 排版原则（大学展示加分项）**
- **对齐**：使用“格式”选项卡中的“对齐”工具（左对齐、居中、横向分布）。
- **对比**：标题大而粗，正文小而细，用颜色突出重点。
- **留白**：别把幻灯片塞满，每页最好不超过5行文字。

**4. 幻灯片母版（高效批量修改）**
- 视图 → 幻灯片母版。
- 在母版中插入Logo、页脚、页码 → 所有幻灯片自动应用。
- 修改标题样式、字体等，无需每页手动改。

**5. 简单动画与切换（适可而止）**
- **动画**：只教“淡入”和“擦除”两种效果，足够大学所有展示场景。
- **切换**：只教“淡出”一种切换效果，统一、干净、不花哨。
- **原则**：标题淡入，正文逐条显示即可。**禁止使用弹跳、旋转、缩放等花哨动画**。

**6. 导出与分享**
- **导出PDF**：文件 → 导出 → 创建PDF（保留布局，不可编辑，适合提交作业）。
- **导出图片**：另存为 → 选择JPEG/PNG → 每页变成一张图（适合插入报告）。

### 二、💡 常用操作技巧
- `F5`：从头放映，`Shift+F5`：从当前页放映。
- 一键对齐：选中多个对象（图片、文本框）→ “格式”→“对齐”。
- 取色器：给形状填充颜色时，用“取色器”吸图片中的颜色，保持配色一致。
- 一键删除所有动画：动画窗格（`Alt+F10` 打开）→ `Ctrl+A` 全选 → 删除。

### 三、🔍 扩展知识点：演讲者视图
- 连接投影仪时，使用“演讲者视图”：幻灯片放映 → 勾选“使用演讲者视图”。
- 你的屏幕显示备注、下一张预览，观众只看幻灯片。

### 四、⚠️ 幽默防坑指南

> 🧨 **坑**：放映时按空格，幻灯片没反应，以为电脑坏了。  
> 👉 **真相**：按 `ESC` 退出，检查是否在放映模式。或者你点到最后一页了。

> 🧨 **坑**：加了一堆动画，放映时观众眼花缭乱，老师说“能不能正常点”。  
> 👉 **真相**：课堂展示不是动画比赛。淡入+擦除足够，少即是多。

> 🧨 **坑**：PPT发微信给老师，老师打不开。  
> 👉 **真相**：微信可能改后缀。另存为 `.pptx` 或直接导出PDF最稳妥。

### 五、✅ 今日练习（45分钟）

- [ ] 制作5页幻灯片：封面、目录、三个内容页（主题“我的家乡”）。
- [ ] 使用母版插入一个简单Logo（可用形状代替）和页码。
- [ ] 练习对齐：将三个图片水平对齐。

## 📅 第9天：Word 论文排版实战加练（综合实战）

> 🎯 **本课目标**：写一篇完整的模拟论文，将第4-5天学的所有技能综合运用，走通全流程。

### 一、📖 实战任务

**任务**：写一篇不少于5页的模拟课程论文，主题自拟（如“大学生时间管理现状分析”或“人工智能对教育的影响”）。

**论文结构要求**：
1. **封面**：标题、姓名、日期、课程名称
2. **目录**：自动生成（使用标题样式）
3. **正文**：
   - 至少3个一级标题（使用“标题1”样式）
   - 每个一级标题下至少2个二级标题（使用“标题2”样式）
   - 正文使用“正文”样式，首行缩进2字符，1.5倍行距
4. **页眉页脚**：
   - 页眉：论文标题
   - 页码：目录页用罗马数字（i, ii, iii…），正文页用阿拉伯数字（1, 2, 3…）
   - 使用分节符实现不同页码格式
5. **表格**：至少插入一个3列以上的表格（如数据统计表），添加表头并重复
6. **图片**：至少插入一张图片，设置为“四周型”环绕
7. **公式**：至少插入一个简单公式（如平均值公式或百分比公式）
8. **参考文献**：至少列出3条（手动输入，统一格式）

### 二、💡 实战操作步骤

1. **新建文档** → 先设置页边距（布局→页边距→适中）
2. **输入标题和正文** → 应用“标题1”“标题2”“正文”样式
3. **插入分节符** → 在目录页与正文之间插入“分节符（下一页）”
4. **设置页码** → 目录部分用罗马数字，正文部分用阿拉伯数字（取消“链接到前一节”）
5. **插入目录** → 引用→目录→自动目录
6. **插入表格和图片** → 设置文字环绕和表头重复
7. **插入公式** → 使用Word自带公式工具
8. **添加页眉** → 插入→页眉→输入论文标题
9. **更新目录** → 右键目录→更新域→更新整个目录
10. **保存** → 同时保存为 `.docx` 和 `.pdf`

### 三、⚠️ 常见卡点提醒

> 🧨 **卡点**：目录页码显示错误。  
> 👉 **解决**：先确认所有标题都正确使用了“标题1”“标题2”样式，然后右键目录→更新域→更新整个目录。

> 🧨 **卡点**：页眉内容在正文和目录页同步变化。  
> 👉 **解决**：分节后，在正文页眉处取消“链接到前一节”，再修改页眉内容。
