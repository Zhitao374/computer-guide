# 第一阶段：电脑基础入门（第1-2天）

## 📅 第1天：电脑基础 + 文件管理入门

### 一、📖 知识点

#### 1. 硬件与软件基础
- **硬件**：看得见摸得着的设备，主机（CPU、内存、硬盘、主板）、显示器、键盘、鼠标。
- **软件**：运行在硬件上的程序，分为两大类：
  - 系统软件：Windows 10/11（电脑操作系统）
  - 应用软件：浏览器、Office、聊天软件等

#### 2. 正确开关机（必修）
- 开机：按下主机电源键 → 等待系统加载，进入桌面。
- 关机：点击左下角**开始菜单** → 选择**电源** → 点击**关机**。
- ⚠️ 重要提醒：**严禁日常使用中直接拔电源、长按强制关机**，会损伤系统与硬件。

#### 3. 鼠标5种基础操作（必修）

| 操作 | 功能作用 |
| ---- | -------- |
| 左键单击 | 选中文件、图标、选项 |
| 左键双击 | 打开文件、文件夹、软件 |
| 右键单击 | 调出快捷功能菜单 |
| 拖拽 | 移动文件、框选内容 |
| 滚轮 | 上下滚动页面、文档 |

#### 4. 键盘基础（必修）

**① 基准键位（盲打定位）**
- 左手：`A S D F`　　右手：`J K L ;`
- 提示：`F` 和 `J` 按键有凸起小点，是手指定位标识。

**② 常用功能键**
- `Shift`：上档键，输入按键上方字符、临时切换中英文
- `Caps Lock`：大写字母锁定开关
- `Backspace`：删除光标**前面**的文字
- `Delete`：删除光标**后面**的文字
- `Enter`：回车键，确认、换行

**③ 文字切换**
- 中英文切换：`Shift` 或 `Ctrl + 空格`
- 中英文标点切换：`Ctrl + .`

#### 5. 盘符与文件存放规则（必修）
- **盘符**：C盘、D盘、E盘，电脑硬盘分区。
- **核心规则**：**个人作业、课件、资料一律存D盘及其他盘，不要存在C盘**。C盘空间不足会导致电脑严重卡顿，重装系统时C盘数据也会全部丢失。
- **桌面**：桌面本质上也是C盘的一部分，**不要把大量文件放在桌面**。

#### 6. 文件与文件夹基础操作（必修）
- **新建文件夹**：在D盘空白处右键 → 新建 → 文件夹 → 命名。
- **文件操作通用快捷键**：
  - `Ctrl + C`：复制
  - `Ctrl + X`：剪切
  - `Ctrl + V`：粘贴
  - `F2`：快速重命名文件/文件夹
- **基础规则**：**删除桌面图标 ≠ 卸载软件**，图标只是快捷入口，软件本体仍在电脑中。

#### 7. 压缩包使用（必修，传作业/课件必备）
- 常见格式：`.zip`、`.rar`、`.7z`
- **解压**：右键压缩包 → 解压到当前文件夹/指定文件夹
- **压缩**：选中多个文件 → 右键 → 发送到 → 压缩(zipped)文件夹，方便批量传文件
- **推荐工具**：7-Zip（免费无广告，大学通用）
- **大学场景**：交小组作业时，把多个文件压缩成一个ZIP，避免漏传。

#### 8. 作业命名规范（必修）
统一格式：**班级+姓名+作业名称**，例：`2026计科1班-张三-高数作业`，符合高校老师要求。

### 二、⚠️ 易错防坑指南

> 🧨 误区：电脑卡顿反复按开机键  
> 👉 解答：普通按键无法加速，长按电源键10秒为**强制关机**，仅死机时应急使用。

> 🧨 误区：删除快捷方式，以为卸载了软件  
> 👉 解答：快捷方式只是“入口”，卸载软件需在系统设置中操作。

> 🧨 误区：把文件、照片全部堆在桌面，C盘飘红  
> 👉 解答：桌面本质在C盘，文件存D盘才是正确习惯。

- [ ] 练习文件复制、剪切、重命名操作
- [ ] 尝试将多个文件压缩、解压

## 📅 第2天：系统快捷键 + 电脑选购 + 新机验机 + 软件卸载

### 一、📖 知识点

#### 1. 高频系统快捷键（必修）
- `Win + E`：一键打开「此电脑」
- `Win + L`：快速锁屏（离开座位必用，保护隐私）
- `Ctrl + Shift + Esc`：直接打开任务管理器（比 Ctrl+Alt+Del 快一步）
- `Ctrl + S`：保存文件（所有软件通用，养成随手保存习惯）

#### 2. 任务管理器基础
- 作用：查看运行程序、内存占用，关闭无响应软件。
- 打开方式：`Ctrl + Shift + Esc` → 选择要结束的程序 → 右键“结束任务”。

#### 3. 大学笔记本选购指南（必修）

**① 核心硬件参考**
- **CPU**：优先选择 Intel Core Ultra 5 / AMD Ryzen 5 及以上，**不选购赛扬、奔腾处理器**（性能过低）
- **内存**：最低 **16GB**，8GB及以下容易卡顿
- **硬盘**：必须选择固态硬盘SSD，容量最低512GB，理工科建议1TB
- **显卡**：文科/商科用集成显卡足够；理工科（编程/绘图/视频剪辑）需独立显卡（RTX 4050及以上）

**② 专业适配建议**
- 文科/商科/教育/法学 → 轻薄本（14寸/15.6寸），集显即可
- 理工科（编程、数据分析）→ 全能本，16G+内存
- 设计/建筑/影视（3D渲染、视频剪辑）→ 游戏本或移动工作站

**③ 避坑清单**
- ❌ 不买8GB以下内存（卡死）
- ❌ 不买128GB硬盘（不够用）
- ❌ 不买赛扬/奔腾/凌动处理器（电子垃圾）
- ❌ 不买无SSD的纯机械硬盘本（开机两分钟）

#### 4. 新机验机要点（新生必用）
- 到货后**先不联网**（跳过WiFi），避免激活系统后无法退换
- 检查外观有无划痕、屏幕有无坏点（可用纯色图片检测）
- 查看硬件配置是否和商品描述一致（任务管理器→性能）
- 确认无问题后，再联网激活系统与Office

#### 5. 标准卸载软件流程（必修）
开始菜单 → 设置 → 应用 → 已安装应用 → 找到对应软件 → 点击卸载。
⚠️ 禁止直接删除软件安装文件夹，会残留垃圾、导致系统异常。

### 二、⚠️ 易错防坑指南

> 🧨 误区：电脑变慢就乱装“加速软件”  
> 👉 解答：多数第三方加速软件为流氓软件，反而拖慢电脑。Windows自带Defender杀毒已够用。

> 🧨 误区：收到新电脑立刻联网激活，后发现屏幕有坏点退不了  
> 👉 解答：验机流程是**先不联网检查 → 确认OK再激活**。

> 🧨 误区：想卸载软件，直接去Program Files文件夹删文件夹  
> 👉 解答：残留的注册表垃圾会拖慢系统，必须走标准卸载流程。

- [ ] 查看电脑已安装软件列表，熟悉卸载入口
- [ ] 对照选购标准，简单判断自己电脑配置水平
