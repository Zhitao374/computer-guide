# 第一阶段 电脑基础入门（第1-3天）
## 📅 第1天：认识电脑 + 鼠标键盘快速上手

### 一、知识点
#### 1. 硬件与软件基础
- **硬件**：看得见摸得着的设备，主机（CPU、内存、硬盘、主板）、显示器、键盘、鼠标。
- **软件**：运行在硬件上的程序，分为两大类
  - 系统软件：Windows 10/11（电脑操作系统）
  - 应用软件：浏览器、Office、聊天软件等

#### 2. 正确开关机（必修）
- 开机：按下主机电源键 → 等待系统加载，进入桌面
- 关机：点击左下角**开始菜单** → 选择**电源** → 点击**关机**
- ⚠️ 重要提醒：**严禁日常使用中直接拔电源、长按强制关机**，会损伤系统与硬件。

#### 3. 鼠标5种基础操作（必修）
| 操作 | 功能作用 |
| ---- | -------- |
| 左键单击 | 选中文件、图标、选项 |
| 左键双击 | 打开文件、文件夹、软件 |
| 右键单击 | 调出快捷功能菜单 |
| 拖拽 | 移动文件、框选内容 |
| 滚轮 | 上下滚动页面、文档 |

#### 4. 键盘基础（必修）
1. **基准键位（盲打定位）**
   左手：`A S D F`  右手：`J K L ;`
   提示：`F` 和 `J` 按键有凸起小点，是手指定位标识。
2. 常用功能键
   - `Shift`：上档键，输入按键上方字符、临时切换中英文
   - `Caps Lock`：大写字母锁定开关
   - `Backspace`：删除光标**前面**的文字
   - `Delete`：删除光标**后面**的文字
   - `Enter`：回车键，确认、换行
3. 文字切换
   - 中英文切换：`Shift` 或 `Ctrl + 空格`
   - 中英文标点切换：`Ctrl + .`
4. 🟡 选学（极少使用）：简繁字体切换，快捷键 `Ctrl+Shift+F`，了解即可。

### 二、易错防坑指南
> 🧨 误区：电脑卡顿反复按开机键
> 👉 解答：普通按键无法加速，长按电源键10秒为**强制关机**，仅死机时应急使用。

### 三、今日课后练习（10分钟 必做）
1. 完整执行一次正常开机、关机操作
2. 练习鼠标单击、双击、拖拽、滚轮操作
3. 打开系统记事本，练习中英文混合输入

## 📅 第2天：系统基础 + 文件管理 + 压缩包使用（大学刚需）

### 一、知识点
#### 1. 系统基础认知
- 桌面：电脑开机后的主界面，图标分为**软件快捷方式**和**文件**
- 盘符：C盘、D盘、E盘，电脑硬盘分区
  - 规则：**个人作业、课件、资料一律存D盘及其他盘，不要存在C盘**，C盘空间不足会导致电脑严重卡顿。
- 文件夹：用于分类存放文件，是大学生整理资料的核心工具。

#### 2. 文件基础操作（必修）
1. 新建文件夹：桌面/空白处右键 → 新建 → 文件夹
2. 文件操作通用快捷键
   - `Ctrl + C`：复制
   - `Ctrl + X`：剪切
   - `Ctrl + V`：粘贴
   - `F2`：快速重命名文件/文件夹
3. 基础规则：**删除桌面图标 ≠ 卸载软件**，图标只是快捷入口，软件本体仍在电脑中。

#### 3. 压缩包使用（必修，传作业/课件必备）
- 常见格式：`.zip`、`.rar`、`.7z`
- 解压：右键压缩包 → 解压到当前文件夹/指定文件夹
- 压缩：选中多个文件 → 右键 → 发送到压缩包，方便批量传文件
- 推荐工具：7-Zip（免费无广告，大学通用）

#### 4. 作业命名规范（必修）
统一格式：**班级+姓名+作业名称**，例：2026计科1班-张三-高数作业，符合高校老师要求。

### 二、易错防坑指南
> 🧨 误区：删除快捷方式，以为卸载了软件
> 👉 解答：快捷方式只是“入口”，卸载软件需在系统设置中操作。

### 三、今日课后练习（12分钟 必做）
1. 在D盘新建3个文件夹：`大学作业`、`课程课件`、`生活资料`
2. 练习文件复制、剪切、重命名操作
3. 尝试将多个文件压缩、解压

## 📅 第3天：系统快捷键 + 电脑选购 + 新机验机 + 软件卸载

### 一、知识点
#### 1. 高频系统快捷键（必修）
- `Win + E`：一键打开「此电脑」
- `Win + L`：快速锁屏（离开座位必用，保护隐私）
- `Ctrl + Shift + Esc`：直接打开任务管理器
- `Ctrl + S`：保存文件（所有软件通用，养成随手保存习惯）

#### 2. 任务管理器基础
作用：查看运行程序、内存占用，关闭无响应软件。

#### 3. 大学笔记本选购指南（必修）
1. 核心硬件参考
   - CPU：优先选择 Intel Core Ultra 5 / AMD Ryzen 5 及以上，**不选购赛扬、奔腾处理器**（性能过低）
   - 内存：最低 **16GB**，8GB及以下容易卡顿
   - 硬盘：必须选择固态硬盘SSD，容量最低512GB，理工科建议1TB
2. 专业适配
   - 文科/商科：轻薄本（14寸/15.6寸），集成显卡即可满足需求
   - 理工科（编程、绘图、视频剪辑、设计）：全能本/游戏本，搭配独立显卡（RTX4050及以上）
3. 避坑清单：不买4GB内存、128GB硬盘、老旧入门级处理器设备。

#### 4. 新机验机要点（新生必用）
1. 到货后先不联网，检查外观、屏幕有无坏点
2. 查看硬件配置是否和商品描述一致
3. 确认无问题后，再联网激活系统与Office

#### 5. 标准卸载软件流程（必修）
开始菜单 → 设置 → 应用 → 已安装应用 → 找到对应软件 → 点击卸载
⚠️ 禁止直接删除软件安装文件夹，会残留垃圾、导致系统异常。

### 二、易错防坑指南
> 🧨 误区：电脑变慢就乱装“加速软件”
> 👉 解答：多数第三方加速软件为流氓软件，反而拖慢电脑。

### 三、今日课后练习（15分钟 必做）
1. 使用快捷键：打开此电脑、锁屏、打开任务管理器
2. 查看电脑已安装软件列表，熟悉卸载入口
3. 对照选购标准，简单判断自己电脑配置水平
