## 📅 第1天：认识电脑 + 鼠标键盘快速上手

### 一、📖 知识点

**1. 硬件与软件**
- **硬件**：主机（CPU、内存、硬盘、主板）、显示器、键盘、鼠标。
- **软件**：操作系统（Windows 10/11）、应用软件（Office、浏览器等）。

**2. 开关机正确步骤**
- 开机：按主机电源按钮 → 等待桌面出现。
- 关机：开始菜单 → 电源 → 关机。**严禁直接拔电源**（相当于让电脑猝死）。

**3. 鼠标5种操作**
| 操作 | 作用 |
|------|------|
| 左键单击 | 选中 |
| 左键双击 | 打开 |
| 右键单击 | 快捷菜单 |
| 拖拽 | 移动/框选 |
| 滚轮 | 滚动页面 |

**4. 键盘打字基础**
- 基准键位：左手 `A S D F`，右手 `J K L ;`（F和J有凸点定位）。
- 常用键：`Shift`（上档）、`Caps Lock`（大写锁定）、`Backspace`（删前）、`Delete`（删后）、`Enter`（回车）。
- 中英文切换：`Shift` 或 `Ctrl+空格`；中文标点按 `Ctrl+.` 切换。
- **简体/繁体切换**：
  - **搜狗输入法**：右键点击输入法状态栏 → 选择“简/繁”切换；或使用快捷键 `Ctrl+Shift+F`。
  - **微软拼音**：点击任务栏右下角的“中”字 → 在弹出菜单中选择“繁”或“简”；或使用快捷键 `Ctrl+Shift+F`。
  - **大学场景**：部分课程需要写繁体作业或阅读繁体文献，学会切换可省去手动转换的麻烦。

### 二、💡 常用操作技巧
- `Win + E`：一键打开“此电脑”。
- `Win + L`：离开时锁屏（防室友偷看）。
- `Ctrl + Shift + Esc`：直接打开任务管理器（比 `Ctrl+Alt+Del` 快一步）。

### 三、🔍 扩展知识点：存储 vs 运行内存
- **运行内存（RAM）**：电脑的“工作台”，临时存放当前运行的程序和数据。**关机会清空**。8GB是起步，16GB大学够用。
- **存储（硬盘）**：电脑的“仓库”，长期存放文件（文档、照片、游戏）。**关机会保留**。256GB SSD 或 512GB 更佳。
- **简单比喻**：RAM是你书桌的桌面大小（桌面大能同时摊开多本书）；硬盘是你的书柜容量（书柜大能存更多书）。
- **为什么电脑卡**：RAM太小，同时开太多程序，桌面放不下了 → 系统用硬盘假装RAM（虚拟内存），速度慢10倍。

**补充：CPU 和 GPU 是什么？**
- **CPU（中央处理器）**：电脑的“大脑”，负责所有计算和逻辑判断。CPU越快，电脑处理任务的速度就越快（打开软件、解压文件、运行程序等）。
- **GPU（图形处理器）**：电脑的“美术老师”，专门负责处理图像、视频、3D渲染。玩游戏、剪视频、做设计时主要靠它。不玩游戏/不做设计的同学，用CPU自带的“集成显卡”就够用了，不需要额外买独立显卡。

> 💡 **选电脑小预告**：大学买电脑，内存要16GB，硬盘必须固态（SSD），CPU别买赛扬/奔腾。更详细的《大学新生选电脑指南（2026版）》请单独查看配套文档。

### 四、⚠️ 幽默防坑指南

> 🧨 **坑**：删除桌面图标，以为卸载了软件。  
> 👉 **真相**：那只是软件的“门牌号”，房子还在。去“设置→应用”卸载。

> 🧨 **坑**：电脑慢就狂按开机键。  
> 👉 **真相**：开机键不是油门，长按10秒是强制关机（相当于打晕电脑）。

> 🧨 **坑**：想打繁体字，结果打出来的是简体，以为输入法坏了。  
> 👉 **真相**：输入法默认是简体模式。在状态栏上右键切换为“繁体”模式，或按 `Ctrl+Shift+F` 即可。
