# 💻 常用快捷键速查表（打印版）

> 建议贴在电脑旁，每天用几次，一周就记住了。

---

## 一、通用操作（最最常用）

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + C` | 复制 |
| `Ctrl + X` | 剪切 |
| `Ctrl + V` | 粘贴 |
| `Ctrl + Z` | 撤销（后悔药） |
| `Ctrl + Y` | 重做（恢复撤销） |
| `Ctrl + A` | 全选 |
| `Ctrl + S` | 保存（随时按！） |
| `Ctrl + F` | 查找（网页/文档中搜关键字） |
| `Delete` | 删除到回收站 |
| `Shift + Delete` | 永久删除（不进回收站，慎用） |

---

## 二、窗口与系统

| 快捷键 | 作用 |
|--------|------|
| `Alt + Tab` | 切换已打开的窗口 |
| `Win + D` | 一键显示桌面（再按恢复） |
| `Win + E` | 打开“此电脑” |
| `Win + L` | 锁屏（离开座位必按） |
| `Win + 数字键` | 打开任务栏第n个程序（如 `Win+1` 打开第一个） |
| `Alt + F4` | 关闭当前窗口 / 关机菜单 |
| `Ctrl + Shift + Esc` | 直接打开任务管理器 |

---

## 三、浏览器（上网）

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + T` | 新建标签页 |
| `Ctrl + W` | 关闭当前标签页 |
| `Ctrl + Shift + T` | 恢复刚关闭的标签页（救急！） |
| `Ctrl + Tab` | 切换到下一个标签页 |
| `Ctrl + D` | 收藏当前网页 |
| `Ctrl + F5` | 强制刷新（不用清理缓存） |

---

## 四、文字编辑（Word/打字）

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + B` | 加粗 |
| `Ctrl + I` | 斜体 |
| `Ctrl + U` | 下划线 |
| `Ctrl + E` | 居中 |
| `Ctrl + L` | 左对齐 |
| `Ctrl + R` | 右对齐 |
| `Ctrl + J` | 两端对齐 |
| `Home` | 光标跳到行首 |
| `End` | 光标跳到行尾 |
| `Ctrl + Backspace` | 删除光标前一个词（而不是一个字） |
| `Ctrl + Shift + 方向键` | 快速选中一段文字 |

---

## 五、截图与输入

| 快捷键 | 作用 |
|--------|------|
| `Win + Shift + S` | 截图（矩形/任意形状/全屏）→ 自动复制到剪贴板 |
| `Win + .`（句点） | 打开表情符号面板 😊🎓 |
| `Win + V` | 剪贴板历史（需在设置中开启） |

---

## 六、记住这几条就够了（入门版）

如果觉得太多，先记这10个：

1. `Ctrl + C` / `X` / `V` —— 复制/剪切/粘贴  
2. `Ctrl + Z` —— 撤销  
3. `Ctrl + S` —— 保存  
4. `Ctrl + F` —— 查找  
5. `Alt + Tab` —— 切换窗口  
6. `Win + D` —— 显示桌面  
7. `Win + E` —— 打开此电脑  
8. `Win + L` —— 锁屏  
9. `Ctrl + A` —— 全选  
10. `Win + Shift + S` —— 截图
