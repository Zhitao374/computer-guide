# 📋 十二天学会电脑 · 课程大纲（速览版）

> 大学新生 · 电脑0基础 · 12天×80分钟（两节课）


| 课次 | 阶段 | 主题 | 核心内容 |
|------|------|------|----------|
| 第1天 | 第一阶段 | 认识电脑 + 鼠标键盘 | 硬件/软件、开关机、鼠标5种操作、键盘基准键位、中英文切换 |
| 第2天 | 第一阶段 | 系统基础 + 文件管理 + 压缩包 | 盘符（文件存D盘）、文件夹操作、复制/剪切/粘贴/重命名、压缩解压 |
| 第3天 | 第一阶段 | 系统快捷键 + 电脑选购 + 验机 | Win+E/L、任务管理器、CPU/内存/硬盘选购标准、新机验机流程、软件卸载 |
| 第4天 | 第二阶段 | Word论文排版基础 | 标题样式、自动目录、页眉页脚、基础页码 |
| 第5天 | 第二阶段 | Word表格、图片与公式 | 表格制作/合并、图片图文混排、截图粘贴、公式插入 |
| 第6天 | 第二阶段 | Excel数据整理与公式 | 自动填充、SUM/AVERAGE/MAX/MIN、单元格格式（文本/货币） |
| 第7天 | 第二阶段 | Excel排序、筛选、透视表与图表 | 排序筛选、数据透视表、柱形图/饼图 |
| 第8天 | 第二阶段 | PPT课堂展示制作 | 主题/版式、母版统一修改、淡入动画/淡出切换、导出PDF |
| 第9天 | 第二阶段 | Word论文综合实战 + 格式兼容 | 论文全流程演练、目录/页眉问题修复、docx+PDF双格式保存 |
| 第10天 | 第三阶段 | 浏览器、搜索、软件管理、校园网 | 标签页快捷键、高级搜索技巧、软件安装避坑、校园网连接、Defender杀毒 |
| 第11天 | 第三阶段 | 电脑卡顿排查、C盘清理、数据备份 | 变慢五步排查法、故障速查表、磁盘清理、U盘安全弹出、本地+云备份 |
| 第12天 | 第四阶段 | 快捷键汇总、邮件、云存储、PDF、在线协作 | 分级快捷键、邮件礼仪、PDF标注/转换、批量重命名、在线协作文档 |


## 全课程总结

✅ 电脑硬件、键鼠、开关机、文件分类管理  
✅ 压缩包、批量重命名、截图等实用小工具  
✅ Word论文排版、Excel数据统计、PPT课堂展示  
✅ 浏览器搜索、软件安全安装、校园网与网课使用  
✅ 电脑故障排查、系统清理、数据安全备份  
✅ 正式邮件、PDF处理、多人在线协作  

**完成12天学习，可完全覆盖大学四年所有电脑使用需求。**
