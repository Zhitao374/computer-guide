# 📋 十二天学会电脑 · 课程大纲（速览版）

> 大学新生 · 电脑0基础 · 12天×80分钟 · Word/Excel交替学习


| 课次 | 主题 | 核心内容 |
|:----:|------|----------|
| 第1天 | 电脑基础 + 文件管理 | 硬件/软件、鼠标键盘、开关机、盘符规则、新建文件夹、复制/剪切/粘贴/重命名、压缩解压 |
| 第2天 | 系统快捷键 + 选购 + 验机 | Win+E/L、任务管理器、CPU/内存/硬盘选购、避坑清单、新机验机、标准卸载软件 |
| 第3天 | Word排版基础 | 标题样式、自动目录、页眉页脚、基础页码、首行缩进 |
| 第4天 | Excel数据与公式 | 自动填充、SUM/AVERAGE/MAX/MIN、单元格格式、前导零问题 |
| 第5天 | Word表格/图片/公式 | 表格制作/合并/重复表头、图片图文混排、截图粘贴、公式插入 |
| 第6天 | Excel排序/筛选/透视表 | 排序筛选、Ctrl+T表格美化、数据透视表分类汇总 |
| 第7天 | Word+Excel综合实战 | Excel分析数据 → Word输出完整报告（首次融合） |
| 第8天 | PPT课堂展示 | 主题/版式、母版、淡入动画/淡出切换、导出PDF |
| 第9天 | Excel图表 + VLOOKUP | 柱形图/饼图、VLOOKUP查找函数、下拉列表（数据验证） |
| 第10天 | Word长文档 + 批注/修订 | 论文全流程排版、批注与修订、docx+PDF双保存 |
| 第11天 | 上网 + 软件 + 维护（合并） | 浏览器/搜索技巧、软件安全/校园网、故障排查/系统清理/数据备份 |
| 第12天 | 效率工具 + 综合复盘 | 分级快捷键、邮件礼仪、云存储、PDF处理、在线协作、自学方法 |


## 全课程总结（一张表）

| 类别 | 学会内容 |
|------|----------|
| 电脑基础 | 硬件/键鼠/开关机、文件分类、压缩包、快捷键、选购验机、软件卸载 |
| Word | 样式/目录/页眉/表格/图片/公式/批注/修订/长文档 |
| Excel | 公式/排序/筛选/透视表/图表/VLOOKUP/数据验证 |
| PPT | 主题/母版/淡入动画/导出PDF |
| 上网维护 | 搜索技巧/软件安全/校园网/故障排查/C盘清理/数据备份 |
| 效率工具 | 邮件礼仪/云存储/PDF处理/批量重命名/在线协作 |

**完成12天学习 = 覆盖大学四年所有电脑使用需求。**
