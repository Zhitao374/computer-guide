# 📁 第三阶段：上网与日常维护（2天）

> 📌 **本阶段目标**：掌握浏览器高效操作、安全下载软件、处理常见电脑故障。


## 📅 第10天：浏览器、搜索、软件管理与安全

### 一、📖 知识点

**1. 浏览器高效操作**
- 常用浏览器：Edge（Windows自带，省心）、Chrome（扩展多）。
- 标签页：`Ctrl+T` 新建，`Ctrl+W` 关闭，`Ctrl+Tab` 切换，`Ctrl+Shift+T` 恢复刚关闭的标签页（救急神器）。
- 书签：`Ctrl+D` 收藏当前网页，定期整理。

**2. 高效搜索技巧（大学生必备）**
- **关键词提取**：用具体词，如“Win10 连接校园网”而非“电脑上不了网”。
- **双引号精确搜索**：`"文献综述模板"` 返回完整短语。
- **限定网站**：`site:edu.cn 数据透视表` 只在教育网站搜索。
- **文件类型**：`filetype:pdf 高等数学习题` 只搜PDF文件。
- **排除关键词**：`苹果 -水果` 排除水果相关内容。

**3. 软件安装与卸载（别中招）**
- **下载渠道**：官网（如 `office.com`）或腾讯软件管家独立版。**避开“高速下载器”**（那会给你装一堆垃圾软件）。
- **安装时**：选“自定义安装”→ **取消所有捆绑软件**（如杀毒、浏览器、输入法）。
- **卸载**：开始菜单 → 设置 → 应用 → 找到软件 → 卸载。**不要直接删除程序文件夹**。

**补充：系统更新设置**
- **要不要更新**：要。Windows 更新会修复安全漏洞和系统Bug，保持更新能防止病毒入侵。
- **什么时候更新**：建议设置“使用时段”（设置 → Windows更新 → 更改使用时段），让更新在夜间或你不用电脑时自动进行。
- **避坑**：不要在赶作业/交论文前点“立即更新”（可能等半小时）。提前规划好时间。

**4. 网络安全基础**
- 不随便点“你的电脑有病毒”、“恭喜中奖”等弹窗。
- 不下载不明来源的 `.exe` 文件，尤其QQ/微信传的。
- 密码管理：不同网站用不同密码，推荐用密码管理器（如Bitwarden）。

**补充：Windows 自带杀毒软件（Defender）已够用**
- **Windows Defender** 是 Windows 自带的杀毒软件，**不需要额外安装** 360、腾讯电脑管家、金山毒霸等。
- 这些“优化软件”不仅拖慢电脑，还会弹广告、捆绑安装其他软件。**大学四年，Defender 完全够用**。
- 如果已经装了，建议卸载掉，只保留 Defender。

### 二、💡 常用操作技巧
- `Win+Shift+S`：截图并自动保存到剪贴板。
- `Win+V`：剪贴板历史（需在设置中开启，可查看多次复制的内容）。
- 浏览器扩展：安装 **uBlock Origin**（广告拦截），浏览网页清爽无比。

### 三、🔍 扩展知识点：校园网与VPN
- **校园网**：通常需要学号认证，注意保管密码，不要接路由器给别人用（可能被封）。
- **VPN**：在校外访问学校图书馆资源（知网、Web of Science）需要VPN。学校官网有教程。

### 四、⚠️ 幽默防坑指南

> 🧨 **坑**：搜索“电脑变慢怎么办”，下载了3个“加速器”，结果电脑更慢了。  
> 👉 **真相**：很多“加速器”是流氓软件。正确方法：清理C盘、禁用开机自启、加内存条。

> 🧨 **坑**：安装某软件时一路“下一步”，结果桌面多了“XX浏览器”和“游戏大厅”。  
> 👉 **真相**：安装时选“自定义安装”，**取消所有勾选**。天下没有免费的午餐。

> 🧨 **坑**：装了一大堆“电脑管家”“杀毒软件”，电脑反而更卡。  
> 👉 **真相**：Windows自带Defender已经够用，装多个杀毒等于请了几个保安互相打架。

## 📅 第11天：常见问题急救与日常维护

### 一、📖 知识点

**1. 电脑变慢的5步法**
1. 打开任务管理器（`Ctrl+Shift+Esc`）→ 查看CPU/内存/磁盘占用高的程序 → 右键“结束任务”。
2. 清理C盘：右键C盘 → 属性 → 磁盘清理 → 勾选临时文件、回收站、下载文件夹。
3. 禁用开机自启：任务管理器 → “启动”选项卡 → 禁用不需要的程序（如音乐播放器、下载工具）。
4. 检查剩余空间：C盘低于10GB会严重变慢，移走个人文件。
5. 重启电脑（解决70%的问题）。

**2. 常见故障速查表**
| 现象 | 解决步骤 |
|------|----------|
| 没声音 | 1. 右下角喇叭是否静音。2. 右键喇叭 → 声音设置 → 确认输出设备。3. 重启。 |
| 连不上Wi-Fi | 1. 右下角网络图标 → 诊断。2. 重启路由器（拔电源10秒）。3. 忘记网络重新连接。 |
| 程序无响应 | 1. 等待几秒。2. `Ctrl+Shift+Esc` → 选中程序 → 结束任务。 |
| 蓝屏 | 1. 手机拍下错误代码。2. 强制重启（长按电源键10秒）。3. 搜索错误代码 + 解决方法。 |
| 鼠标/键盘没反应 | 检查USB插头，换个口。或者重启电脑。 |
| **补充：电脑发热、风扇噪音大** | 1. 检查底部通风口是否被堵（不要放在被子、抱枕上）。2. 用笔记本支架垫高，帮助散热。3. 用任务管理器查看是否有程序占用CPU过高，结束任务。4. 如果持续发热，可能是灰尘太多，去维修店清灰。 |

**3. 数据备份（重要！！！）**
- **原则**：重要文件至少存两份在不同设备。
- **备份方案**：
  - 本地备份：D盘 + 外接硬盘/U盘。
  - 云备份：百度网盘（2T免费）、OneDrive（5G免费）、阿里云盘（不限速）。
- **大学场景**：论文、作业、照片定期备份，U盘易丢，云盘更可靠。

### 二、💡 常用操作技巧
- `Win + R` → 输入 `%temp%` → 删除临时文件（安全，可删）。
- 查看电脑配置：右键“此电脑” → 属性（看CPU和内存），任务管理器 → 性能（看磁盘类型）。
- 系统还原点：重要操作前（如装驱动），可创建还原点（控制面板 → 恢复 → 配置系统还原）。

### 三、🔍 扩展知识点：固态硬盘的保养
- **不要碎片整理**：SSD不需要传统磁盘碎片整理（会缩短寿命）。Windows会自动识别并只做“优化”。
- **保留20%空间**：SSD越满速度越慢，建议剩余空间>20%。
- **开启TRIM**：默认开启，确保删除文件后SSD能真正清空。

### 四、⚠️ 幽默防坑指南

> 🧨 **坑**：蓝屏后疯狂按键盘，屏幕没反应。  
> 👉 **真相**：蓝屏时系统已死，唯一能做的：长按电源键10秒强制关机，再开机。

> 🧨 **坑**：清理C盘时，把 `Windows` 文件夹里的东西删了，然后电脑开不了机。  
> 👉 **真相**：别碰系统文件夹！只用“磁盘清理”工具，别手贱。

> 🧨 **坑**：U盘直接拔掉，下次插入提示“需要扫描修复”。  
> 👉 **真相**：U盘也会“内伤”。安全弹出再拔（右键U盘盘符 → 弹出）。

> 🧨 **坑**：电脑发烫，以为坏了，拿风扇对着吹。  
> 👉 **真相**：电脑发热是正常现象，但过热会降频变慢。先检查通风口，垫高底座，不行再考虑清灰。
