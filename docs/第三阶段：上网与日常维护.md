# 第三阶段：上网安全与日常维护（第11天）


## 📅 第11天：浏览器、搜索、软件管理、校园网、故障排查（合并）

### 一、📖 知识点

#### 1. 浏览器基础（推荐Edge/Chrome 必修）
- 标签页快捷键：`Ctrl+T`新建标签、`Ctrl+W`关闭标签、`Ctrl+Shift+T`恢复关闭的标签（救急神器）
- 网页收藏：`Ctrl+D` 添加书签

#### 2. 高效搜索技巧（大学生必备 必修）
- 提炼精准关键词，拒绝模糊描述（“Win11连接校园网” vs “电脑上不了网”）
- 双引号 `\"内容\"`：精确匹配完整短语
- `filetype:pdf`：只搜索PDF文档
- `site:edu.cn`：仅在高校网站搜索资料

#### 3. 软件安装避坑（必修）
- 下载渠道：官方网站、正规软件平台，**拒绝高速下载器**
- 安装设置：选择「自定义安装」，**取消所有捆绑软件勾选**
- 卸载流程：沿用第2天标准卸载方式

#### 4. 校园网络连接（必修）
- 有线网：插入网线 → 浏览器弹出认证页 → 输入学号密码登录
- 无线网：连接校园WiFi名称，完成网页认证
- 提示：不同学校流程略有差异，可提前在新生群咨询

#### 5. 网课与校园打印（必修）
- 网课工具：学习通、腾讯会议、雨课堂（网页端基础操作）
- 校园打印：文档/PDF上传至校园打印系统，在线打印

#### 6. 网络安全基础（必修）
- 不点陌生弹窗、中奖提示、病毒警告
- 不运行QQ/微信接收的未知 `.exe` 文件
- 系统防护：Windows自带Defender杀毒软件足够使用，**不额外安装多款电脑管家**

#### 7. 电脑变慢五步排查法（必修 按顺序操作）
1. **重启电脑**（解决70%小问题，最简单最先试）
2. **任务管理器结束高占用程序**（`Ctrl+Shift+Esc`）
3. **磁盘清理**：右键C盘 → 属性 → 磁盘清理
4. **保证C盘剩余空间大于10GB**（空间不足会严重卡顿）
5. **禁用开机自启**：任务管理器→启动项，禁用不需要的程序

#### 8. 高频故障速查（必修）

| 故障现象 | 解决方法 |
| ---- | ---- |
| 无声音 | 检查静音、切换输出设备、重启 |
| 连不上WiFi | 网络诊断、重启路由器、忘记网络重连 |
| 程序无响应 | 任务管理器结束任务 |
| 蓝屏 | 拍照记录代码，长按电源键强制重启 |
| 键鼠失灵 | 重新插拔USB接口、重启电脑 |
| 电脑发热 | 垫高笔记本、保持通风，关闭高占用程序 |

#### 9. 系统清理与数据备份（必修）
- **临时文件清理**：`Win+R` 输入 `%temp%`，全选删除（安全操作）
- **U盘使用**：使用完毕**右键弹出**，禁止直接拔插
- **数据备份原则**：重要文件**多设备备份**，不单独存U盘
  - 本地备份：电脑D盘 + U盘/移动硬盘
  - 云备份：阿里云盘、百度网盘、OneDrive

### 二、⚠️ 易错防坑指南

> 🧨 误区：一路点击“下一步”安装软件，捆绑大量垃圾程序  
> 👉 解答：必须选择自定义安装，取消捆绑勾选。

> 🧨 误区：随意删除C盘Windows系统文件夹  
> 👉 解答：仅使用系统自带磁盘清理工具，不动系统目录。

> 🧨 误区：U盘直接拔下，导致文件损坏  
> 👉 解答：必须执行“安全弹出”后再拔。
