# 第三阶段 上网安全与日常维护（第10-11天）
## 📅 第10天：浏览器、搜索技巧、软件管理、校园网、网课工具

### 一、知识点
#### 1. 浏览器基础（推荐Edge/Chrome 必修）
- 标签页快捷键：
  `Ctrl+T`新建标签、`Ctrl+W`关闭标签、`Ctrl+Shift+T`恢复关闭的标签
- 网页收藏：`Ctrl+D` 添加书签

#### 2. 高效搜索技巧（大学生必备 必修）
1. 提炼精准关键词，拒绝模糊描述
2. 双引号 `\"内容\"`：精确匹配完整短语
3. `filetype:pdf`：只搜索PDF文档
4. `site:edu.cn`：仅在高校网站搜索资料

#### 3. 软件安装避坑（必修）
- 下载渠道：官方网站、正规软件平台，**拒绝高速下载器**
- 安装设置：选择「自定义安装」，**取消所有捆绑软件勾选**
- 卸载流程：沿用第3天标准卸载方式。

#### 4. 校园网络连接（必修）
- 有线网：插入网线 → 浏览器弹出认证页 → 输入学号密码登录
- 无线网：连接校园WiFi名称，完成网页认证
- 提示：不同学校流程略有差异，可咨询学长/辅导员。

#### 5. 网课与校园打印（必修 新增刚需）
- 网课工具：学习通、腾讯会议、雨课堂（网页端基础操作）
- 校园打印：文档/PDF上传至校园打印系统，在线打印。

#### 6. 网络安全基础（必修）
- 不点陌生弹窗、中奖提示、病毒警告弹窗
- 不运行QQ/微信接收的未知 `.exe` 文件
- 系统防护：Windows自带Defender杀毒软件足够使用，**不额外安装多款电脑管家**（占用资源、弹广告）。

### 二、易错防坑指南
> 🧨 误区：一路点击“下一步”安装软件，捆绑大量垃圾程序
> 👉 解答：必须选择自定义安装，取消捆绑勾选。

### 三、今日课后练习（12分钟 必做）
1. 练习标签页新建、关闭、恢复操作，收藏常用网页
2. 搜索本校校园网连接教程，熟悉流程

## 📅 第11天：电脑卡顿排查、常见故障、C盘清理、数据备份

### 一、知识点
#### 1. 电脑变慢五步排查法（必修 按顺序操作）
1. 重启电脑（解决70%小问题）
2. 任务管理器结束高占用程序
3. 右键C盘 → 属性 → 磁盘清理
4. 保证C盘剩余空间大于10GB
5. 任务管理器→启动项，禁用不必要的开机自启软件

#### 2. 高频故障速查（必修）
| 故障现象 | 解决方法 |
| ---- | ---- |
| 无声音 | 检查静音、切换输出设备、重启 |
| 连不上WiFi | 网络诊断、重启路由器、忘记网络重连 |
| 程序无响应 | 任务管理器结束任务 |
| 蓝屏 | 拍照记录代码，长按电源键强制重启 |
| 键鼠失灵 | 重新插拔USB接口、重启电脑 |
| 电脑发热 | 垫高笔记本、保持通风，关闭高占用程序 |

#### 3. 系统清理（必修）
- 临时文件清理：`Win+R` 输入 `%temp%`，全选删除文件（安全操作）
- U盘使用：使用完毕**右键弹出**，禁止直接拔插，避免文件损坏。

#### 4. 数据备份（重中之重 必修）
原则：重要文件**多设备备份**，不单独存U盘
- 本地备份：电脑D盘 + U盘/移动硬盘
- 云备份：阿里云盘、百度网盘、OneDrive

#### 5. 🟡 选学：固态硬盘保养、系统还原点（了解即可）

### 二、易错防坑指南
> 🧨 误区：随意删除C盘Windows系统文件夹
> 👉 解答：仅使用系统自带磁盘清理工具，不动系统目录。
> 🧨 误区：U盘直接拔下，导致文件损坏
> 👉 解答：必须执行“安全弹出”后再拔。

### 三、今日课后练习（15分钟 必做）
1. 执行C盘磁盘清理、删除系统临时文件
2. 查看任务管理器，识别高占用程序
3. 有U盘则练习安全弹出操作
